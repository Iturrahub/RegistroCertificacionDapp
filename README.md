# Registro de Certificados Blockchain

## Descripción

Este proyecto corresponde a la Evaluación Final Transversal de la
asignatura **Fundamentos de Blockchain**.

La aplicación consiste en una **DApp (Aplicación Descentralizada)** que
permite emitir, verificar y revocar certificados académicos mediante un
contrato inteligente desarrollado en **Solidity** e implementado sobre
la red **Ethereum Sepolia**.

El objetivo del proyecto es demostrar el uso de blockchain para
garantizar la integridad, autenticidad y trazabilidad de la información
mediante contratos inteligentes.

## Tecnologías utilizadas

-   Solidity
-   Remix IDE
-   Ethereum Sepolia
-   MetaMask
-   HTML5
-   CSS3
-   JavaScript
-   ethers.js
-   GitHub

## Funcionalidades

-   Conexión de usuarios mediante MetaMask.
-   Emisión de certificados académicos.
-   Verificación de certificados por ID.
-   Verificación de certificados mediante hash.
-   Revocación de certificados por el administrador.
-   Registro permanente de la información en blockchain.

## Requisitos

-   MetaMask instalada.
-   Cuenta conectada a la red Ethereum Sepolia.
-   Navegador web moderno.
-   Live Server (VS Code) u otro servidor local.

## Ejecución

1.  Desplegar el contrato inteligente en Remix IDE sobre Sepolia.
2.  Copiar la dirección del contrato.
3.  Actualizar la variable `contratoDireccion` en `app.js`.
4.  Abrir `index.html` mediante Live Server.
5.  Conectar MetaMask y utilizar las funcionalidades de la aplicación.
